Task Payment Tracker PWA files

Upload these files and folders to the root of your existing GitHub repository:

1. index.html
2. manifest.webmanifest
3. service-worker.js
4. icons folder, including all icon PNG files

GitHub upload steps:

1. Open your GitHub repository.
2. Click Add file > Upload files.
3. Drag all files from this folder into GitHub, including the icons folder.
4. If GitHub asks whether to replace index.html, allow it. This updated index.html includes the PWA links.
5. Click Commit changes.
6. Wait one to three minutes for GitHub Pages to update.
7. Open your GitHub Pages app link in Safari on iPhone.
8. Refresh the page once.
9. Tap Share > Add to Home Screen.
10. Open the new Home Screen app icon.

After opening the Home Screen app once while online, it should continue to open offline using the cached files. Your task data is still stored locally in the browser/app storage on that device.

To update the app later:

1. Upload the newer files to GitHub.
2. Open the app in Safari.
3. Refresh once or twice.
4. Open from the Home Screen again.

Important:

Keep your JSON sync backup somewhere safe. The hosted files do not store your task data in GitHub.
