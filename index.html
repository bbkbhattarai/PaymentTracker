First draft notes

This app is designed to work without hosting. It stores records in the browser you use. Use the Device Sync JSON section to move the full app data between your phone and laptop. Previously entered names, emails, roles, task titles and locations are suggested as you type.

Setup and travel claims

Use the Setup button to save your name, address, bank details and mileage rate. If travel is claimable for a task, choose Yes and add the miles. The app calculates the travel amount and itemises it in the payment request.

Task calendar entry

The calendar button downloads an .ics file for the task only. Payment follow-up is handled inside the dashboard, not in your calendar.

Email/payment flow

After completion, generate a payment request, then choose the general format or the Club Academy Scotland referee claim form. Use Share, Download, Print/Save PDF, or Open Email Draft.

Generate payment request

Your name, address, bank details and mileage rate are pulled from Setup. The Club Academy form uses task title as Match and job title as Type of Official.

For phone use, Share may attach the generated request in supported apps. If not, use Download HTML or Print/Save PDF, then attach manually.

Export tasks

Device sync JSON

Use this when switching between phone and laptop. Export the full sync JSON from the device with the newest data, send it to yourself by AirDrop, email, WhatsApp, iCloud Drive, Google Drive or cable transfer, then import it on the other device. The file includes tasks, setup details and bank details, so share it only through trusted routes.

Merge keeps the newest version of each task using its updated timestamp. If you deleted a task on one device and want the deletion reflected everywhere, use Replace from the device that has the correct final list.
